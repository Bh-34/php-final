Resumo do meu projeto

Projeto web para compra e gerenciamento de cafés: frontend em Vue.js e backend em Laravel que permite cadastro/login de usuários, visualização do catálogo de cafés, criação de pedidos que são enfileirados para preparo e acompanhamento; inclui painel administrativo para gerenciar cafés e pedidos, scripts e seeders para inicializar dados, migrations para o banco e rotas/API para o frontend. É empacotado com Vite, e configurações de filas, cache e e-mail, sendo modular e fácil de manter e expandir.
